setting up the project.

Installation.
This project uses Node.js and mongodb.
 1. To install mongodb follow instructions here: http://docs.mongodb.org/manual/installation/
 2. Install node.js from http://nodejs.org
 

 Run.
 1. First of all database is need to be initialized. In order to do it, run load("inserts.js") in mongo shell. Normally we use moon_db for tests and sun_db for deployment.
